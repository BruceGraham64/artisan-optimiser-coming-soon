- logo.png         : Placeholder logo (replace with your final logo if needed).
- favicon.png      : Browser icon.
- readme.txt       : Deployment instructions.

How to Deploy on Vercel (Using CLI):
1. Install Node.js (https://nodejs.org) if not already installed.
2. Open Command Prompt or PowerShell.
3. Install the Vercel CLI:
   npm install -g vercel
4. Log in:
   vercel login
5. Extract this ZIP and navigate to its folder:
   cd path/to/extracted/folder
6. Deploy to Vercel:
   vercel
   (Choose 'No' when asked about linking to a project. Name it e.g. artisan-optimiser)
7. Make it live:
   vercel --prod
8. Link your domain:
   Go to https://vercel.com → Project → Settings → Domains → Add artisanoptimiser.com and follow DNS instructions.
